  ⚙️ Реализация

В программе реализованы следующие функции:
•	login(username, password) — проверка корректности входа
•	get_username_status(username) — определение статуса пользователя
•	is_admin(username) — проверка, является ли пользователь администратором
Логика работы:
1.	Пользователь вводит имя и пароль
2.	Выполняется проверка входа
3.	Определяется статус пользователя
4.	Результат выводится в консоль

   
  ▶️ Пример выполнения

Введите имя: admin
Введите пароль: 1234
Статус: Администратор
